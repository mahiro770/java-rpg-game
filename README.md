# java-rpg-game
継承を用いたrpgゲームのsakusei 
- Java

## ■ クラス構成
- Hero（勇者）
- SuperHero（勇者の拡張）
- Matango（モンスター）
- PoisonMatango（毒攻撃を持つモンスター）

## ■ 工夫した点
- 継承を使い、SuperHeroで機能拡張
- メソッドのオーバーライドで振る舞いを変更
- クラスごとの責務を分けて設計

## ■ 今後の改善
- UI（画面表示）の追加
- ダメージ計算のロジック強化
- アイテム機能の追加
